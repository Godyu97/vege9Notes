3. redis实现分布式锁

   * set mux uuid nx ex  上锁并设置过期时间

   * del mux 释放锁 
   * 使用uuid表示不同的操作，释放锁时判断uuid是否正确，防止误删；不是原子操作，使用lua脚本保证原子性
   * 条件：
     * 互斥性，只有一个客户端能持有锁
     * 不会发生死锁，某客户端不主动解锁，也能保证后续运行
     * 加锁解锁必须是同一客户端
     * 加锁解锁具有原子性
